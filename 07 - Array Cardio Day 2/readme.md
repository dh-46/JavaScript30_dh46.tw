## 筆記

### some() & every()

#### ``.some()``

- 陣列中只要有一個元素符合你要的條件，就會是true。

#### ``.every()``

- 陣列中必須每個元素都符合條件，才會是true。

### find()

- 類似filter的作用，但是只會回傳第一個符合你查詢的item。
